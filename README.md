# 设计初衷
常见的素材管理器满足不了自己的需求，希望能有一个可以高自由度可定制的文件管理器

# 功能列表
- [x] sqlite3存储结构，允许自定义关联表
- [x] 没有限制的扩展插件，事件广播多
- [x] 模块化设计，各个功能互不干扰
- [x] 标签页设计，切换更便捷
- [x] 内置全局呼出搜索
- [x] 代码热更新
- [ ] 前后端分离，支持部署私人素材库
- [ ] ...
